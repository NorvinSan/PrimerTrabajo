Requisitos de Visual Studio Code

  • 1.6 GHz o un procesador más rápido
  • 1 GB de RAM o más.
  • Microsoft .NET Framework 4.5.2 es necesario para Visual Studio Code. Si es usuario de Windows 7, asegurarse de tener .NET Framework 4.5.2 instalado.
  • Tarjeta de vídeo compatible con DirectX 9 con resolución de pantalla de 1024 x 768 o más..
